HOW-TO

Create a button showing the number of times it has been clicked.

DESCRIPTION

Counting the number of button clicks using JavaScript and jQuery can be easily implemented using the instructions below:
(Note: You are free to use your favorite text editor)

SOLUTION

How to create a button counter for our project using jQuery:

Getting started: For this demo we are going to use the jQuery CDN. Insert the following line at the bottom of the body tag:

<script src="https://code.jquery.com/jquery-2.1.4.min.js"></script>

  1. Create a blank html page.
  2. Create a button with an id attribute set to any value: In our case let it be "button-counter":

    <button id="button-counter"></button>

  3. Open a script tag OR create a blank JavaScript file and reference it in the page, after the jQuery reference,
    and create a variable that will represent the counter:
    (Note: You might want to consider storing this variable in your database or web storage so that its value can be used even after the browser has been closed,
    since in this case it will be reset back to zero.)

    var counter = 0;

  4. Select the button using jQuery and set its text to the initial value of the counter variable:

    var buttonCounter = $('#button-counter');
    buttonCounter.text(counter);

  5. Attach a click event to the button increasing the value of the counter and setting the new value as the button's text:

    buttonCounter.on('click', function() {
        counter += 1;
        buttonCounter.text(counter);
    });
